# Labels3D Extension for GDevelop 5

The Labels3D extension allows you to add labels to objects or positions.

The extension offers three actions:
> Create label
> 
> Update label position by ID
> 
> Delete label by ID

These labels can be customized: you can change their color, add or remove a shadow, and change the shadow color. Any characters can be used in the text.
